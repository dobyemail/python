DobyEmail Class Descriptions

Service: Main class that encapsulates both SMTP and IMAP operations, providing a high-level interface for sending and receiving emails.

Sender: Handles the composition and sending of emails, including support for attachments.

Reader: Manages connections to IMAP servers, fetching emails, and parsing their content.

Transfer: Facilitates the movement of emails between different email accounts or folders.

Config: Manages configuration settings for the email service, including server addresses, ports, and authentication details.

Functions:

parse_date: Converts various date string formats to a standardized format.

check_ports: Verifies if specific ports are open on a given host.

is_valid_email: Provides basic validation for email addresses.

get_email_ports: Retrieves standard email port configurations from the email_ports.json file.
